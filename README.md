# Noncepad Tap

## How do I install these formulae?

`brew install noncepad/tap/<formula>`

Or `brew tap noncepad/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "noncepad/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
